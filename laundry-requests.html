🔗 Enable live sharing across devices

Right now requests only save on this device. Follow these steps once to share with your whole team in real time — it's free!

▶ Show setup steps
  1. Go to console.firebase.google.com and sign in with a Google account
  2. Click "Add project" → name it (e.g. "Hotel Laundry") → click through to create it
  3. Inside your project, click "Realtime Database" in the left menu → "Create database" → choose Start in test mode → Done
  4. Click the ⚙️ gear → "Project settings" → scroll to "Your apps" → click the </> web icon → Register app → copy the firebaseConfig block
  5. Open this HTML file in a text editor, find the line that says PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder with your config
  6. Save the file and open it in your browser — the orange banner will disappear and the green dot will say Live ✓
// It will look like this — paste yours in place of it: const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "hotel-laundry.firebaseapp.com", databaseURL: "https://hotel-laundry-default-rtdb.firebaseio.com", projectId: "hotel-laundry", storageBucket: "hotel-laundry.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc..." };

Then host the file anywhere (email it, put it on a shared drive, or upload to netlify.com/drop for a free link you can send to the whole team.)
